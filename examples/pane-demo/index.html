Pane Component Demo

This example demonstrates the Pane component with three different panes that have custom z-index values:

Background Pane (z-index: 350): Blue circles below default overlay pane

Middle Pane (z-index: 550): Red markers between shadows and default markers

Foreground Pane (z-index: 650): Green polygons above default marker pane

Use the checkboxes below to toggle each pane on/off and observe how the z-index affects the stacking order. All elements overlap at the center to clearly show layering effects.
